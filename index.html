📦 What is UNSPSC?

The United Nations Standard Products and Services Code (UNSPSC) is a global taxonomy used to classify all products and services. By standardizing procurement data into 8-digit codes, organizations achieve complete spend visibility, automate purchasing analytics, and optimize global supply chains.

Historically, mapping messy industrial part descriptions to the correct 8-digit code has been incredibly tedious and error-prone. This tool solves that by using Agentic AI to autonomously research and classify parts in seconds.

🧠 How Decisions are Made

1. Web Search Refinement

Queries DuckDuckGo via proxy. The raw search results are distilled by the LLM into a concise summary of the product's Industry, Use Case, and Category.

2. Agentic Classification & Reflection

At each level (Segment → Family → Class → Commodity), the model selects the best match. Crucially, a secondary Auditor LLM immediately reviews the choice. If the auditor rejects the classification, the agent halts to prevent deep hallucinations and suggests an alternative path.

3. Strict Traceability

The model is mathematically restricted using JSON Schema Enums to prevent hallucination. Expand "🔍 Debug Trace" on any card to audit the Prompt and Output.
